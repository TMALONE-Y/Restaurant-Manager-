A robust, CLI-based Restaurant Management System designed for multiple owners. This application provides a secure environment for users to manage their restaurant portfolios with real-time data persistence.

---

## 🛠️ Key Features

* **👤 Multi-User Architecture:** Supports independent user accounts. Each owner has their own isolated data space.
* **🔒 Secure Authentication:** Built-in login and registration system to protect user data.
* **📂 Persistent Data Storage:** Leverages JSON for lightweight, file-based database management, ensuring data is never lost.
* **⚡ Full CRUD Functionality:** Seamlessly **C**reate, **R**ead, **U**pdate, and **D**elete restaurant records.
* **🗑️ Account Control:** Users have full authority to manage or permanently delete their profiles and associated data.

---

## 🚀 How It Works

1.  **Authentication:** Start by creating a new account or logging into an existing one.
2.  **User Dashboard:** Access a personalized menu to manage your restaurants.
3.  **Data Sync:** Every action (adding or deleting) is instantly synchronized with `system_data.json`.
